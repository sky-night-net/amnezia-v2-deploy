# 🛡️ SkyKnight Amnezia VPN Automator v2.0

Автоматизированный инструмент для развертывания VPN-сервера на базе протокола **Amnezia WG 1.5** с кастомной премиальной панелью управления (Premium Dashboard) и умной обфускацией.

## 🚀 Основные возможности

*   **One-Click Deploy:** Полная настройка чистого сервера (Ubuntu) за один клик.
*   **Auto-Docker:** Если Docker не установлен на сервере, приложение установит его автоматически.
*   **Premium Web UI:** Замена стандартного интерфейса на современный "Glassmorphism" Dashboard с графиками и статистикой.
*   **Smart Download:** Все токены обфускации (`I1`, `I2`) автоматически прописываются в скачиваемые `.conf` файлы.
*   **Статистика:** Встроенная система мониторинга живого трафика по каждому клиенту.
*   **Автономность:** Весь необходимый софт и дизайн уже упакованы в приложение.

## 📥 Установка (macOS)

1. Скачайте последнюю версию `SkyKnightAutomator.dmg` из раздела [Releases] (или соберите сами).
2. Перетащите в папку "Программы" (Applications).
3. Запустите (при первом запуске может потребоваться разрешение в Системных настройках -> Защита и безопасность).

## 🛠️ Как пользоваться

1. Введите **IP-адрес** вашего нового сервера.
2. Введите **SSH-пароль** (пользователь `root`).
3. (Опционально) Настройте **Токены обфускации**, если хотите сменить маскировку.
4. Нажмите **DEPLOY NETWORK**.
5. Дождитесь завершения (логи будут в нижнем окне).
6. После успеха перейдите по ссылке `http://IP:4455`, используя ваш мастер-пароль.

## 🔨 Сборка из исходников

Если вы разработчик:
1. Клонируйте репозиторий.
2. Установите зависимости: `pip install -r requirements.txt`.
3. Запустите сборку: `./build_macos.sh`.
