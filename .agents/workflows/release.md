---
description: Автоматический процесс подготовки релиза (Bump version & Commit)
---

Этот процесс автоматизирует рутинную задачу выпуска новой версии.
Выполняй следующие шаги:

1. Проверь файл `VERSION`.
2. Увеличь версию на 0.0.1 (например, с 1.3.1 на 1.3.2).
// turbo
3. Обнови файл `VERSION` с новым номером.
4. Сообщи пользователю: "Версия успешно обновлена до [новая версия]".
// turbo
5. Сделай коммит в Git с сообщением "chore: bump version to [новая версия]".
6. Сделай Push в репозиторий.
7. Подтверди завершение: "Релиз подготовлен и отправлен на GitHub! 🚀"
