# 🛡️ Amnezia VPN CLI v2 — Premium Deployment Suite

**Продвинутый комплекс для управления VPN на базе AmneziaWG (v1.5+). Включает в себя Premium Dashboard, систему аналитики трафика и кроссплатформенный Automator.**

---

## 🚀 Что нового в v2?

В отличие от классической версии, **v2** предлагает полноценную инфраструктуру:
- **Premium Dashboard**: Современный Glassmorphism Web-интерфейс для управления сервером.
- **Automator App**: macOS Desktop приложение для развертывания серверов в 1 клик.
- **Live Statistics**: Нативный `statsCollector` с базой SQLite для мониторинга живого трафика (RX/TX) каждого клиента.
- **Smart Obfuscation**: Продвинутая обфускация (JC, JMIN, JMAX, S1, S2, H1-H4 + **I1/I2 токены**).

---

*   **Master Dashboard**: Единый веб-интерфейс для переключения между серверами и просмотра общей аналитики.

## ⚡ Установка (Terminal App)

Интерфейс командной строки стал максимально простым и кроссплатформенным. Выберите вашу операционную систему:

### 🪟 Windows (PowerShell)
Для Windows работает нативный автоматический установщик, который сам загрузит необходимые зависимости (Git, Python) и настроит окружение.
1. Нажмите `Win + X` и выберите **Windows PowerShell** (или Терминал).
2. Скопируйте, вставьте и выполните следующую команду:
```powershell
irm https://raw.githubusercontent.com/sky-night-net/amnezia-v2-deploy/main/install.ps1 | iex
```
*В дальнейшем для запуска программы просто введите команду `amneziav2` в терминале.*

### 🐧 Linux (Ubuntu / Debian / CentOS / Arch)
Вставьте эту команду в ваш терминал:
```bash
curl -sL https://raw.githubusercontent.com/sky-night-net/amnezia-v2-deploy/main/install.sh | bash
```
*В дальнейшем для запуска программы просто введите команду `amneziav2` в терминале.*

### 🍎 macOS
Вставьте эту команду в стандартное приложение **Terminal.app**:
```bash
curl -sL https://raw.githubusercontent.com/sky-night-net/amnezia-v2-deploy/main/install.sh | bash
```
*(Установщик полностью поддерживает Homebrew и автоматически установит недостающие компоненты).*

> **💡 Оффлайн запуск и Автообновления:**
> После первой установки (с интернетом), все графические и логические узлы приложения устанавливаются в вашу систему глобально (в `~/.amnezia-v2`). Теперь вы можете управлять вашими серверами локально командой `amneziav2` без интернета! А при выходе новой версии вы можете обновиться прямо через пункт **«Проверить обновления»** в меню программы.

### 🛡️ Как работает автоматическая установка? (Transparent Init)
Мы уважаем вашу безопасность. Когда вы запускаете `install.sh` (Linux/macOS) или `install.ps1` (Windows PowerShell), скрипт предсказуемо делает 5 вещей:
1. **Анализ среды**: Проверяет, установлены ли `Python 3` и `Git`.
2. **Безопасная установка**: Если чего-то не хватает, он честно предупреждает пользователя, спрашивает разрешение и загружает нативные зависимости через ваш официальный системный пакетный менеджер (`winget`, `APT`, `YUM`, `Homebrew`). Никаких сторонних `.exe`!
3. **Клонирование кода**: Загружает исключительно актуальный открытый исходный код с данного GitHub репозитория в вашу закрытую домашнюю папку `~/.amnezia-v2`.
4. **Установка Python-модулей**: Ставит только популярные модули с официального PyPI ресурса (`paramiko` для SSH-соединения с серверами, `bcrypt` и `requests`).
5. **Создание шортката**: Создает исполняемую ссылку или `.bat` файл для удобного и глобального вызова команды `amneziav2` из любого терминала в мире, без лишних окон.

Если вы являетесь пользователем macOS и желаете получить полноценное самостоятельное Desktop-приложение вместо интерфейса командной строки:

2. Перейдите в папку: `cd amnezia-v2-deploy/Automator_App`
3. Запустите сборку: `./build_macos.sh`
4. Готовое нативное приложение **SkyKnightAutomator.app** появится в папке `dist/`.

## 📊 Premium Dashboard и Статистика
Панель управления будет доступна по указанному вами при установке порту (по умолчанию `4466`).
Аналитика трафика собирается нативно напрямую из пространства ядра WireGuard и доступна по API на порту `9191`.

## 🛠️ Структура проекта
- `amnezia-cli.py` — Интерактивный мастер развертывания.
- `amnezia-deploy.py` — Ядро развертывания Docker контейнера.
- `Automator_App/` — Исходный код macOS интерфейса (Tkinter).

- `stats/` — Автономный скрипт сбора статистики в SQLite базу данных.
